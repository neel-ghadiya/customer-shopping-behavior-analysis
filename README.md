# customer-shopping-behavior-analysis

End-to-end data analytics project analyzing customer shopping behavior using Python, SQL, and Power BI to uncover insights on spending patterns, customer segmentation, and product performance.

## 🛍️ Customer Shopping Behavior Analysis 📌 Project Overview

This project analyzes customer shopping behavior using transactional data (3,900 records) to uncover insights into:

*   **Customer spending patterns**
*   **Product preferences**
*   **Subscription behavior**
*   **Customer segmentation**

The goal is to support data-driven business decisions using a complete analytics pipeline.

## 🤖 Tech Stack

*   **Python (Pandas, NumPy)** – Data cleaning & EDA
*   **SQL (PostgreSQL)** – Business analysis queries
*   **Power BI** – Interactive dashboard
*   **Jupyter Notebook** – Analysis workflow

---

### 📊 Dataset Summary
*   **Rows:** 3,900
*   **Columns:** 18
*   **Features include:**
    *   **Demographics:** Age, Gender, Location
    *   **Purchase Details:** Category, Amount, Season
    *   **Behavior:** Discount, Frequency, Reviews

---

### 📈 Key Analysis (SQL)
To uncover the business logic, I wrote queries to answer the following:
*   **Revenue Analysis:** By gender and age group
*   **Customer Segmentation:** Categorized users as New, Returning, or Loyal
*   **Behavioral Insights:** Comparing Subscriber vs. Non-subscriber spending
*   **Product Performance:** Identifying top-rated products and shipping type comparisons

---

### 📊 Dashboard (Power BI)
I built an interactive dashboard to visualize the findings:
*   **Revenue Trends** and customer segments
*   **Product Performance** and subscription insights

---

### 💡 Key Insights & 🚀 Business Recommendations
*   **High Value:** Subscribers and loyal customers contribute significantly more to total revenue.
*   **Strategy:** I recommend improving subscription offerings and introducing targeted loyalty programs.
*   **Marketing:** Marketing efforts should focus on high-value segments and optimizing discount strategies for specific products.
